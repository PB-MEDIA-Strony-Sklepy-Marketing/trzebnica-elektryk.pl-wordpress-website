## Self-Service Resources

### Common Issues

**Site is slow:**
1. Clear WordPress cache (WP Super Cache)
2. Check Wordfence scan results
3. Review server resources

**Contact form not working:**
1. Check Contact Form 7 settings
2. Verify SMTP configuration (WP Mail SMTP)
3. Check spam folder for test emails

**Images not loading:**
1. Verify uploads directory permissions (755)
2. Check .htaccess file
3. Clear browser cache

**404 errors after updates:**
1. Go to Settings → Permalinks
2. Click "Save Changes" (flushes rewrite rules)
